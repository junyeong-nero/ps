```
leetcode/programmers 디렉토리 아래에 카테고리 별로 분류되지 않은 *.py 를 확인하고 아래와 같은 테스크를 진행해줘.

1. 문제를 유형별로 분류하여 카테고리 별로 *.py 파일을 옮김.
2. 분류한 문제를 logs/ 디렉토리에 YYYY-MM-DD.md 형식으로 작성. 
 - 이 마크다운 파일에는 문제에 대한 유형이 작성되어야 해. 
 - 최대한 간단하게 "문제번호: 풀이방식 및 사용 데이터 구조" 형식을 사용해
3. 업데이트된 문제들을 하나씩 commit, 마지막으로 작성한 로그도 commit.
 - 문제를 커밋할 때는 "[solve] 문제이름" 형식의 commit message style 을 적용해줘.
 - 로그를 커밋할 때는 "[summary] YYYY-MM-DD" 형식의 commit message style 을 적용해줘.

```
